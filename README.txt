=== OpenID Header ===
Tags: openid, identifiers, identity
Contributors: mjgiarlo

Inserts a <link> tag into each WordPress post and page that claims ownership of the resource by a specified OpenID.  This is explained quite thoroughly in Sam Ruby's blog - http://www.intertwingly.net/blog/2007/01/03/OpenID-for-non-SuperUsers

== Downloading ==

1. Grab the zip file from http://www.lackoftalent.org/michael/blog/openid-for-wordpress/
2. Or check it out of svn: svn co http://svn.wp-plugins.org/openid/trunk openid

== Installation ==

1. Upload the openid folder containing all source files to your plug-ins folder, usually `wp-content/plugins/`
2. Login as an administrator and activate the plug-in via the Plugins menu
3. Enter your OpenID and an OpenID server via the OpenID submenu of the Options menu.  
4. Voila!  Your blog is OpenID-enabled.
